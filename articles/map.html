STL Map

Ce este un Map din STL ?

  • În C++ Standard Template Library (STL), containerul std::map este un tip de structură de date asociativă ce permite stocarea perechilor cheie-valoare. Fiecare cheie este unică, iar valorile asociate acestor chei pot fi accesate, adăugate sau modificate rapid. Elementele dintr-un std::map sunt stocate ordonat, conform ordinii definite de chei, care utilizează, implicit, operatorul operator< pentru a compara elementele. std::map este implementat folosind o structură de arbore binar echilibrat (de obicei, un arbore AVL sau Red-Black), ceea ce asigură că operațiile de inserare, ștergere și căutare au complexitate logaritmică O(log n).
  • Acesta se declară astfel : map<tip_data,tip_data> nume_map;
  • Este necesară #include <map> pentru a putea utiliza structura de date si metodele sale.

Operații și funcții aplicate pe Map

  • nume_map[nume_tip_data_1] = nume_tip_data_2 Atribuie la cheia cu identificatorul nume_tip_data_1(cheia) variabila nume_tip_data_2(valoarea) (Relație cheie-valoare).
  • nume_map.erase(nume_cheie) Șterge perechea cheie-valoara.
  • nume_map.find(nume_cheie) Caută perechea cheie-valoare dupa cheia transmisă ca parametru :
    auto it = nume_map.find(nume_cheie);
    if(it != nume_map.end()){ /// Daca este egal cu iteratorul end atunci se află in afara structurii.
        cout << "Cheia există și valoarea este: " << it->second; /// Afisăm valoarea corespunzatoare cheii
    }
  • nume_map.size() Returnează dimensiunea structurii de date(In principiu, numărul de perechi cheie-valoare atribuite).

Deque în STL

  • std::deque este utilizat pentru gestionarea unei colecții de date în care este necesară inserarea sau ștergerea rapidă de elemente la ambele capete (în față și în spate).
  • Acesta se declară astfel : deque<tip_data> nume_coadă;

Operații și funcții aplicate pe Deque

  • Deque dispune de iteratori si acces aleator al elementor, spre deosebire de queue.
  • nume_coadă.push_back(variabila) Adaugă la finalul cozii variabila introdusă intre paranteze.
  • nume_coadă.push_front(variabila) Adaugă la începutul cozii variabila introdusă intre paranteze.
  • nume_coadă.pop_back() Sterge ultimul element.
  • nume_coadă.pop_front() Sterge primul element.
  • nume_coadă.front() Furnizează primul element.
  • nume_coadă.back() Furninează ultimul element.
  • nume_coadă.size() Returnează numărul de elemente.
  • nume_coadă.clear() Elimină toate elementele din coadă.
  • nume_coadă.find(first, last, value) Verifică dacă value există între iteratorii first și last.
  • nume_coadă.empty() Verifică dacă coada este goală. Returnează true dacă coada este goală, altfel false.
  • nume_coadă.erase(iterator) Sterge elementul prezent la poziția iteratorului transmis ca parametru.
  • Parcurgerea este identică cu coada normală, la care se adauga parcurgerea prin iteratori.(Aceasta poate fi facută din ambele capete).

Priority Queue în STL

  • std::priority_queue este utilizat pentru gestionarea unei colecții de date în care contează ordinea elementelor. Aceasta se bazează pe structura de date Heap.
  • Acesta se declară astfel : priority_queue <tip_data> nume_coadă; sau :
  • priority_queue <tip_data, container_adiacent, functie_comparare> nume_coadă;
  • Containerul adiacent este de obicei vector din STL iar functia de comparare poate fi facută de mână. (Eu folosesc in special greater<tip_data>)
  • priority_queue <int, vector<int>, greater<int>> q; - Elementele sunt sortate crescător.

Operații și funcții aplicate pe Priority_queue

  • nume_coadă.push(variabila) Adaugă în funcție de criteriul de sortare elementul varabila in coadă.
  • nume_coadă.top() Furnizează elementul din vârf.(Cel mai din stânga daca reprezentăm liniar).
  • nume_coadă.size() Returnează numărul de elemente.
  • nume_coadă.clear() Elimină toate elementele din coadă.
  • nume_coadă.empty() Verifică dacă coada este goală. Returnează true dacă coada este goală, altfel false.
  • Parcurgerea este identică cu coada normală.

Unde putem aplica aceste cozi ?

  • Algoritmul Fill cu coadă.
  • Algoritmul lui Lee cu coadă.
  • Two Pointers cu coadă.
  • BFS - Breadth First Search.
  • Algoritmul lui Dijkstra.
  • Sliding Window method (De obicei cu deque).
