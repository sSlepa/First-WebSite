STL Stack

Ce este un Stack din STL ?

  • Stack-ul din STL este o structură de date liniară de tip FILO (First In Last Out) care permite operația de introducere de date pe la un capăt si operația de stergere a datelor la același capăt.
  • Acesta se declară astfel : stack<tip_data> nume_stivă;
  • Este necesară #include <stack> pentru a putea utiliza structura de date si metodele sale.

Operații și funcții aplicate pe Stack

  • nume_stivă.push(variabila) Adaugă la finalul stivei variabila introdusă intre paranteze.
  • nume_stivă.pop() Sterge ultimul element adăugat(de la final / dreapta daca privim structura liniar).
  • nume_stivă.size() Returnează numărul de elemente.
  • nume_stivă.top() Furninează valorea din varful stivă.
  • nume_stivă.clear() Elimină toate elementele din stivă.
  • nume_stivă.empty() Verifică dacă stiva este goală. Returnează true dacă stiva este goală, altfel false.

Din păcate, parcurgerea se poate realiza intr-un singur mod :

stack<int> s;
for(int i = 0 ; i < 5 ; ++i)///Pentru a usura adaugam primele 5 numere in vector
    s.push(i);

while(!s.empty()){
    cout << s.top() << ' ';
    s.pop();
}

Unde putem aplica aceste stive ?

  • Problema gestionării vagoanelor de tren.
  • Sortarea topologică.
  • Ajută la găsirea componentelor tari conexe (Algoritmul lui Kosaraju).
  • Analiza amortizată (Nivel de națională).
