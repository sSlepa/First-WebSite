STL Vector

Ce este un Vector din STL ?

  • Vectorul din STL este un container dinamic, secvențial și liniar ce-și poate modifica dimensiunea in timpul execuției (alocare dinamică)
  • Vectorii își gestionează singuri alocarea memoriei. Când vectorul crește, alocă mai multă memorie, iar când se micșorează, deallocatează memoria după necesități.
  • Acesta se declară astfel : vector<tip_data> nume_vector(N,X);
  • N reprezintă numărul de elemente iar X valorea cu care se umple vectorul. Acestea pot să lipsească nefiind necesare.

Operații și funcții aplicate pe Vector

  • nume_vector.push_back(variabila) adaugă la finalul vectorului variabila introdusă intre paranteze.
  • nume_vector.pop_back() sterge de la finalul vectorului un element
  • nume_vector.size() returnează numărul de elemente ale vectorului
