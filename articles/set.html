STL Set

Ce este un Set din STL ?

  • În C++ Standard Template Library (STL), set este un container asociativ care stochează elementele într-o ordine sortată și fără duplicate. Fiecare element este unic, iar containerul menține ordinea elementelor utilizând o structură de arbore binar echilibrat (de obicei, un arbore Red-Black). Datorită acestei structuri, operațiile de căutare, inserare și ștergere se realizează în timp logaritmic O(log n).
  • Acesta se declară astfel : set<tip_data> nume_set;
  • Este necesară #include <set> pentru a putea utiliza structura de date si metodele sale.
